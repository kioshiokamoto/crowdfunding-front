## Ejecutar localmente

Clonar proyecto

```bash
  git clone https://github.com/kioshiokamoto/crowdfunding-front
```

Ir a ruta de proyecto

```bash
  cd crowdfunding-front
```

Instalar dependencias

```bash
  yarn
```

Iniciar servidor

```bash
  yarn start
```
